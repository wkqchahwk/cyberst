# Incident Response

## Overview

English reference guidance for incident response.

## Workflow

1. Define the target, scope, and constraints.
2. Identify the highest-value test surface for this topic.
3. Validate findings with reproducible evidence.
4. Summarize impact, uncertainty, and remediation.

## Output

- Key evidence
- Practical impact
- Recommended fix
- Safe next steps

## Safety

- Work only in authorized environments.
- Use the minimum-impact validation needed to prove the issue.
